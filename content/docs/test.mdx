## Agent

Agent is the core concept of OxyGent, representing an intelligent entity with specific capabilities and goals.

```python
from oxygent import Agent

# Create a new agent
agent = Agent(
    name="assistant",
    description="An intelligent assistant",
    capabilities=["text_generation", "code_analysis"]
)
```
