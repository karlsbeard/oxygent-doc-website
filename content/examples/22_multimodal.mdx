## Overview

This example demonstrates multimodal in OxyGent.

## Code

```python
import os

from oxygent import MAS, oxy

oxy_space = [
    oxy.HttpLLM(  # Register multimodal LLM
        name="default_vlm",
        api_key=os.getenv("DEFAULT_VLM_API_KEY"),
        base_url=os.getenv("DEFAULT_VLM_BASE_URL"),
        model_name=os.getenv("DEFAULT_VLM_MODEL_NAME"),
        is_multimodal_supported=True,  # Declare multimodal LLM
        is_convert_url_to_base64=True,  # Auto convert to base64 encoding
    ),
    oxy.ReActAgent(
        name="master_agent",
        llm_model="default_vlm",  # Configure multimodal LLM
    ),
]

async def main():
    async with MAS(oxy_space=oxy_space) as mas:
        await mas.start_web_service(first_query="What is this")  # Upload image attachment from frontend

if __name__ == "__main__":
    import asyncio

    asyncio.run(main())
```
