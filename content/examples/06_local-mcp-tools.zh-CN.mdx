## 概述

本示例演示如何在 OxyGent 中使用本地 MCP 工具。

## 代码

```python
"""
依赖 ./mcp_servers/jd_docs_local.py 文件
"""
import os
from oxygent import MAS, oxy

oxy_space = [
    oxy.HttpLLM(
        name="default_llm",
        api_key=os.getenv("DEFAULT_LLM_API_KEY"),
        base_url=os.getenv("DEFAULT_LLM_BASE_URL"),
        model_name=os.getenv("DEFAULT_LLM_MODEL_NAME"),
        llm_params={"temperature": 0.1},
    ),
    oxy.StdioMCPClient(   # 注册LocalMCP工具
        name="jd_docs_tools",
        params={
            "command": "uv",
            "args": ["--directory", "./mcp_servers", "run", "jd_docs_local.py"],
        },
    ),
    oxy.ReActAgent(
        name="QA_agent",
        llm_model="default_llm",
        tools=["jd_docs_tools"],   # 给Agent装配工具
    ),
]

async def main():
    async with MAS(oxy_space=oxy_space) as mas:
        await mas.start_web_service(first_query="京东的211时效是什么？")

if __name__ == "__main__":
    import asyncio
    asyncio.run(main())
```

## 要点

- 本示例展示如何使用 OxyGent 使用本地 MCP 工具
- 请按照代码注释获取详细说明
- 运行前请确保已设置好环境变量

## 运行示例

运行此示例：

