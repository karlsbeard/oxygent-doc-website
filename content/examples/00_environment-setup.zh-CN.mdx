## 概述

本示例演示如何在 OxyGent 中进行环境安装。

## 代码

```python
# python环境（3.10及以上版本）
"""
conda create -n oxy_env python==3.10
conda activate oxy_env
"""

# oxygent环境
"""
pip install oxygent
"""

# Node.js环境（如果使用MCP工具） 可以选择https://nodejs.org/zh-cn下载安装
"""
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
\. "$HOME/.nvm/nvm.sh"
nvm install 22
"""
```

## 要点

- 本示例展示如何使用 OxyGent 进行环境安装
- 请按照代码注释获取详细说明
- 运行前请确保已设置好环境变量

## 运行示例

运行此示例：

```bash
python 00_环境安装.py
```
