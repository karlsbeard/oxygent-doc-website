## Overview

This example demonstrates environment setup in OxyGent.

## Code

```python
# python环境（3.10及以上版本）
"""
conda create -n oxy_env python==3.10
conda activate oxy_env
"""

# oxygent环境
"""
pip install oxygent
"""

# Node.js环境（如果使用MCP工具） 可以选择https://nodejs.org/zh-cn下载安装
"""
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
\. "$HOME/.nvm/nvm.sh"
nvm install 22
"""
```

## Key Points

- This example shows how to use OxyGent for environment setup
- Follow the code comments for detailed explanations
- Make sure to set up your environment variables before running
