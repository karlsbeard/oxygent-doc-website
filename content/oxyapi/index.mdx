## Quick Start

If you're new to the OxyGent API, we recommend starting with the following steps:

1. Check out [ReactAgent API](/oxyapi/agents-react-api) to understand basic agent usage
2. Learn [HttpLLM API](/oxyapi/llms-http-api) to understand how to configure large language models
3. Explore [FunctionHub API](/oxyapi/tools-function-hub-api) to understand the tool system

## API Conventions

All OxyGent APIs follow these conventions:

- **Async Operations**: Most API methods are asynchronous and require the `await` keyword
- **Type Safety**: All APIs provide complete type annotations
- **Error Handling**: API methods throw specific exception types for easier error handling
- **Flexible Configuration**: Support multiple configuration methods, including parameter passing and configuration files
